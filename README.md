# SOPALE Mantle Convection Modeling

## Overview
This repository presents a 2D numerical modeling study of mantle convection and subduction dynamics using SOPALE. The project focuses on simulating lithosphere–mantle interactions and understanding the physical processes governing subduction behavior.

The work demonstrates end-to-end capability in computational geodynamics — from model development and simulation to post-processing, visualization, and scientific interpretation.

---

## Objectives
- Simulate mantle convection and subduction dynamics in a 2D framework
- Investigate lithosphere–mantle interaction processes
- Analyze thermal and dynamic evolution of subducting systems
- Generate publication-quality visualizations and diagnostics

---

## Methodology

### Numerical Modeling
- Code: **SOPALE (Fortran-based geodynamic modeling framework)**
- Model type: 2D mantle convection / subduction system
- Approach:
  - Setup of initial and boundary conditions
  - Definition of rheological structure
  - Time-dependent simulation of mantle flow

### Post-Processing & Analysis
- **GMT (Generic Mapping Tools)**  
  - Spatial visualization  
  - Map generation  
  - Field plotting  

- **MATLAB & Python**  
  - Data extraction and processing  
  - Quantitative analysis  
  - Graph generation  

### Visualization & Figure Design
- **Adobe Illustrator & Inkscape**  
  - Refinement of figures  
  - Creation of publication-quality diagrams
